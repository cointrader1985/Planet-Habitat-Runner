# Planet-Habitat-Runner
Movement across different living environments starting from a home, transitioning through an apartment, and expanding toward a broader conceptual view of earth exploration. Each stage represents a progression system driven by random events and player choices.

The project demonstrates loops, conditionals, randomness, and state updates in a clean structure. It is designed to be easy to read, modify, and extend for beginners learning C++.

Developers can expand the system by adding inventory, energy systems, or multi-path exploration routes. The code is intentionally minimal to encourage experimentation.
